# Kazesift Search (札幌カフェ検索)

札幌エリアのカフェを検索・発見するためのモダンなWebアプリケーションです。  
"作業のしやすさ"、"コーヒーの美味しさ"、"雰囲気" などの独自の基準でカフェを探すことができます。

**URL:** [https://kazesift-search.vercel.app](https://kazesift-search.vercel.app)

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

## 🌟 特徴 (Features)

### 1. 高度な検索・フィルタリング
*   **特徴タグ検索**: Wi-Fi, 電源, 静か, 夜カフェ, 絶景など、具体的なニーズに合わせて絞り込みが可能。
*   **エリア検索**: 大通, 札幌駅, 円山など、主要エリアごとにカフェを探せます。
*   **並び替え**: 評価順、新着順などでリストをソート可能。

### 2. マップ表示 (Leaflet Map)
*   札幌市内のカフェを地図上で確認できます。
*   住所から座標を推定するロジックを搭載し、正確な位置情報を視覚化。

### 3. 詳細なカフェ情報
*   **Kazesift Comment**: 独自の視点によるカフェの魅力を紹介。
*   **営業情報**: 営業時間、定休日、Google Mapsへのリンク。
*   **匿名コメント機能**: ユーザーが感想を書き込める掲示板機能（ローカル保存）。

### 4. インフルエンサー / キュレーター紹介
*   「Kazesift」が信頼するインフルエンサー（Curators）のおすすめカフェを紹介するセクション。
*   SNSへのリンクと、彼らの "Good Points" を掲載。

### 5. その他の機能
*   **お気に入り機能**: 気になるカフェをブックマーク（ブラウザ保存）。
*   **完全レスポンシブ**: スマートフォン、タブレット、PCのすべてのデバイスに最適化。
*   **Google Analytics**: アクセス解析を統合済み。
